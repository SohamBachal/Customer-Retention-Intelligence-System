# Customer Retention Intelligence System

## Objective
Analyze telecom customer churn and identify revenue-risk customer segments.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## Key Features
- Churn analysis
- Revenue loss analysis
- Customer segmentation
- Risk scoring
- Business recommendations

## Key Insights
- Month-to-month contracts have highest churn
- High-value customers contribute major revenue leakage
- Short-tenure premium customers are high risk
